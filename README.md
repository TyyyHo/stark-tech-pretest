## Getting Started

1. 安裝依賴

```bash
npm install
```

2. 本地啟動

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

3. 訪問頁面

[http://localhost:3000](http://localhost:3000)
